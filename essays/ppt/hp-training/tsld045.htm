Which API?

  • thread - traditional Python API

      import thread

      thread.start_new(doit, (5,))
      # (can't easily wait for its completion)

  • threading - resembles Java API

      from threading import Thread # and much more...

      t = Thread(target=doit, args=(5,))
      t.start()
      t.join()
