When to catch exceptions

  • When there's an alternative option

      try:

      f = open(".startup")
      except IOError:
      f = None # No startup file; use defaults

  • To exit with nice error message

      try:

      f = open("data")
      except IOError, msg:
      print "I/O Error:", msg; sys.exit(1)
