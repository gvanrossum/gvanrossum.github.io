High-level properties

  • Extremely portable

    • Unix, Windows, Mac, BeOS, Win/CE, DOS, OS/2, Amiga, VMS, Cray, …

  • Compiles to interpreted byte code

    • compilation is implicit and automatic

  • Memory mgt through ref counting

    • easier for C/C++ extensions

  • “Safe”: no core dumps
