Dictionaries

  • Hash tables, "associative arrays"

    • d = {"duck": "eend", "water": "water"}

  • Lookup:

    • d["duck"] -> "eend"
    • d["back"] # raises KeyError exception

  • Delete, insert, overwrite :

    • del d["water"] # {"duck": "eend", "back": "rug"}
    • d["back"] = "rug" # {"duck": "eend", "back": "rug"}
    • d["duck"] = "duik" # {"duck": "duik", "back": "rug"}
